# Feuille de route

Objectif : suivre plusieurs heures de cours sans revenir à une autre application.

- [x] matières, chapitres et pages ;
- [x] sauvegarde automatique et restauration ;
- [x] stylo, gomme, surligneur, formes et zoom ;
- [x] rejet de paume logiciel ;
- [x] import et annotation PDF ;
- [x] blocs mathématiques colorés ;
- [ ] test longue durée sur Xiaomi Pad ;
- [ ] amélioration de la latence et du lissage ;
- [ ] lasso libre robuste ;
- [ ] réorganisation par glisser-déposer ;
- [ ] miniatures de pages plus lisibles ;
- [ ] tests automatisés de sauvegarde/import/export.

## Critère de passage en version stable

La version 1.0 stable devra réussir un test réel de cours de deux heures sans perte de données, sans blocage de l'éditeur et avec export de sauvegarde fonctionnel.
