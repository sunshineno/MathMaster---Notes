# MathMaster Notes

Application web de prise de notes manuscrites au stylet pour les cours de mathématiques.

## Première version

- Matières et chapitres
- Écriture au stylet
- Stylo, gomme, épaisseur et couleur
- Sauvegarde locale sur l'appareil
- Export PNG
- Impression / export PDF
- Installation comme application web

La reconnaissance manuscrite et la conversion automatique en LaTeX seront ajoutées dans une prochaine version.
