# MathMaster Notes — version 0.3

Cette version ajoute l'environnement LaTeX personnel de MathMaster Notes.

## Nouveautés

- export d'un chapitre complet en fichier `.tex` ;
- préambule LaTeX intégré ;
- environnements Définition, Théorème, Proposition, Lemme, Exercice, Correction et Démonstration ;
- boutons d'insertion rapide de blocs LaTeX ;
- génération automatique du titre, de la table des matières, du chapitre et des sections ;
- conservation des fonctions manuscrites de la version 0.2.

## Déploiement

Remplacer les anciens fichiers du dépôt par ceux de cette archive puis effectuer un commit sur `main`.
GitHub Actions reconstruira automatiquement le site.
