# MathMaster Notes

Application web progressive de prise de notes manuscrites, pensée pour les étudiants en mathématiques.

## Fonctionnalités de la version 0.2

- matières et chapitres ;
- plusieurs pages par chapitre ;
- écriture au stylet avec pression ;
- stylo, gomme, couleur, épaisseur et annulation ;
- fonds blanc, quadrillé, pointillé ou ligné ;
- favoris ;
- recherche de chapitres ;
- sauvegarde locale ;
- export PNG ;
- zone LaTeX par page ;
- déploiement automatique sur GitHub Pages.

## Développement local

```bash
npm install
npm run dev
```

## Déploiement GitHub Pages

Dans **Settings → Pages**, choisir **GitHub Actions** comme source. Chaque modification envoyée sur la branche `main` déclenche ensuite automatiquement le déploiement.
