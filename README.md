# MathMaster Notes

Version stable 1.1.0.

## Fonctions principales

- matières, chapitres et pages ;
- dessin au stylet, surligneur, gomme, formes et sélection ;
- rejet de paume logiciel ;
- sauvegarde automatique, export et restauration ;
- import et annotation PDF ;
- blocs mathématiques colorés ;
- export PDF et LaTeX ;
- installation PWA.

## Développement local

```bash
npm install
npm run build
npm run dev
```
