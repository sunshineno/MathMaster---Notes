INSTALLATION
1. Décompresser ce ZIP.
2. Copier son contenu à la racine du dépôt MathMaster---Notes.
3. Accepter le remplacement des fichiers.
4. Ne pas remplacer package-lock.json.
5. Vérifier avec : npm run build
6. Commit conseillé : v4.0.0 - Texte clavier et espace LaTeX complet
