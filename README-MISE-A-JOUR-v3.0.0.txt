MathMaster Notes v3.0.0 — Espace de travail mathématique

Nouveautés :
- page extensible jusqu’à 9600 px ;
- ajout de sections de travail sans créer une nouvelle page ;
- les formules LaTeX deviennent de vrais objets sur le canevas ;
- déplacement, modification, suppression et redimensionnement des objets LaTeX ;
- sauvegarde locale et cloud des nouveaux objets ;
- compatibilité avec les anciens cahiers.

Installation : copiez les fichiers du patch à la racine du dépôt. Ne remplacez pas package-lock.json. Puis exécutez npm install et npm run build.
